# Software Examples


## Controlling 128 NeoPixels (8x16 Matrix)

This example demonstrates how to control a matrix of 128 WS28B20 (NeoPixel-compatible) RGB LEDs arranged in an 8x16 grid. The code shows how to initialize the matrix, set colors, and display scrolling text.

### Features

- Simple text display on an 8x16 NeoPixel matrix
- Basic color and brightness control
- Example code for quick deployment
